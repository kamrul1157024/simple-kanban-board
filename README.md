# Simple Kanban Board

### Build With:

CSS Grid, ReactJS, ReactDnD

<img src="./simple-kanban-board.gif">

### TO run:
```
npm start
```
### lint:
```
npm run lint
```

### Data:
Your data will control the view.Change the data, view will happen accordingly,
```
data = [
  {
    colName: 'QuoteList1',
    color: 'purple',
    colId:'......' ,
    data: [
            {
                text:'........',
                rowId:'......'
            }
            .......
            .......
            .......
        ],
  },
  .....
  .....
  .....
];
```
